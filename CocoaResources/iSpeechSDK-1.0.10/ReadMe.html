iSpeech SDK for iOS

This is a new iOS SDK, re-written from scratch. There are major changes. It is highly recommended that you check out the documentation to see what all has been changed, and for instructions on integrating with the new SDK. If you run into any bugs, report them in our iPhone SDK Forum.

What’s in this folder?

Compatibility Class

This is a class for you to use while migrating from the old SDK to the new SDK. It takes the new SDK, and wraps it in the API of the old SDK.

Documentation

The documentation for the new SDK’s classes and protocols, as well as a Programming Guide.

Headers

The headers for the SDK.

iSpeechSample

A sample project that uses the SDK. Use it as a reference to get up and running with the SDK.

iSpeechSDK.bundle

The resources used by the SDK.

libiSpeechSDK.a

The actual SDK. The SDK is provided as universal binary, which means that one library works for both the device and the simulator.
